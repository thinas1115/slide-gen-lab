# 本文用画像

`type: "image"`で大きく配置するPNG/JPEGを置く。

- 生成AIで作成した画像、利用者提供画像、利用許諾を確認したWeb画像を保存する。
- `content.json`では`"image": "images/example.png"`のように指定する。
- 推奨解像度は16:9で1600x900以上。縦長・正方形画像も`fit: "contain"`で利用できる。
- Web画像は利用条件を確認し、スライドの`source`と、リポジトリへ同梱する場合は
  `../CREDITS.md`へ出典・ライセンスを記録する。
- 機密画像や再配布できない画像はコミットせず、利用環境ごとに配置する。
