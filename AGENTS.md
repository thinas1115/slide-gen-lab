# AIエージェント共通ルール

このリポジトリで作業するAIエージェントは、作業開始時に `CLAUDE.md` を読み、そこにある
開発フロー・品質ゲート・設計原則を絶対ルールとして扱うこと。

## 記述言語

- 人が読む開発成果物と運用記録は原則として日本語で記述する。
- Issue・PR・レビュー・コミット・進捗報告・完了報告・説明ドキュメントを、英語の定型文で
  作成しない。
- コード識別子、コマンド、パス、正式な製品名、ログ原文など必要な英語だけを例外とし、
  その説明は日本語で書く。
- ユーザーが明示的に別言語を指定した場合のみ、その指定を優先する。

詳細と正本は `CLAUDE.md` を参照すること。
